## Download

Go to [Releases](https://github.com/6ky/volari-releases/releases/latest) and download the latest installer for Windows.

After installing, activate your license from [volari.app](https://volari.app).

## Features

- Schedule and automate Instagram reposts
- Bulk caption generation
- Multi-account session management
- Proxy support with IP verification
- Discord webhook notifications
- Auto-updates

## Requirements

- Windows 10 or later
- An active Volari license
